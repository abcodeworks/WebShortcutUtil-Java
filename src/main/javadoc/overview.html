A Java library for reading and writing web shortcut files.

This module is part of the WebShortcutUtil suite. For more details see the main website.

See the {@link com.abcodeworks.webshortcututil.read read} and {@link com.abcodeworks.webshortcututil.write write} packages for usage details.

A brief list of the supported shortcut types:

  • .desktop - Free Desktop shortcut (used by Linux)
  • .url - Used by Windows
  • .website - Used by Windows
  • .webloc - Used by Apple/Mac/OSX
